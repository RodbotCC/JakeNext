# Channels vs Other External Connection Modes

Use channels when:
- an external system should push into the session already running on your machine
- Claude should react with your real local files and current state in view

Use Claude Code on the web when:
- you want a fresh cloud sandbox

Use standard MCP when:
- Claude should query a system on demand rather than receive pushed events

Use Remote Control when:
- you want to drive your local session remotely from Claude apps
