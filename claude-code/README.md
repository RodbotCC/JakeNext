# Claude Code Library

This directory is the Claude Code shelf.

It is intentionally separate from [capabilities](/Users/jakeaaron/Documents/Storie/capabilities), which now serves as the Anthropic API/SDK capability library we built earlier.

## Split

`capabilities/`
- Anthropic platform and SDK mechanics
- tool APIs
- implementation patterns for building with Claude programmatically

`claude-code/`
- Claude Code product features
- CLI mechanics
- agent workflows
- subagents, teams, memory, permissions, and other CC-specific behavior

## Format

Claude Code entries are intentionally smaller and tighter.

Most of these can be:
- one focused directory per topic
- one main README
- one small manifest
- optional examples only when they add real value
